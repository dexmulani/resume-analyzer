# AI Resume Analyzer

AI-powered Resume Analyzer with ATS scoring, skill extraction, and resume parsing.

-------------

## Features

- ATS Score Detection
- Resume Parsing
- Skill Extraction
- Job Description Matching
- Resume Analysis Dashboard   

---

## Tech Stack

- JavaScript
- Node.js
- HTML/CSS

- Express.js
